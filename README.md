# Guess the Number 🎯

A simple command-line Python game where the player tries to guess a randomly generated number within a limited number of attempts.

## How it works
- The program picks a random number within a set range.
- The player enters guesses.
- After each guess, the program tells you if your guess was too high, too low, or correct.
- The game ends when you guess correctly.

## How to run
1. Make sure Python is installed (python --version to check).
2. Clone this repository:
   git clone https://github.com/Sowmyakuruva19/Guess_number.git
3. Navigate into the folder:
   cd Guess_number
4. Run the script:
   python guess_number.py
